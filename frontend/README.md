# RecipeGenAI Frontend

Modern React + Vite frontend for the multi-agent recipe recommendation system.

## Quick Start

```bash
cd frontend
npm install
cp .env.example .env   # optional: customize API URL
npm run dev
```

App runs at `http://localhost:5173`.

## Features

- Recipe generation form with cuisine, diet, and time preferences
- Real-time loading animation during multi-agent processing
- Nutrition cards, ingredient lists, and step-by-step instructions
- Save recipes to local storage
- Recipe history tracking
- Download recipe as PDF
- Missing ingredient suggestions and shopping list
- Fully responsive mobile-friendly design

## Tech Stack

- React 19
- Vite 8
- Tailwind CSS 4
- Axios
- jsPDF
